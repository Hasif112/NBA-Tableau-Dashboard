# NBA Scoring Trends Analysis (1980–2017)

## Overview
Interactive Tableau dashboard analyzing how NBA basketball has evolved over 37 years using 13,000+ real player seasons.

## Live Dashboard
🔗 [View on Tableau Public](https://public.tableau.com/app/profile/haris.asif/viz/Book1_17755858696270/Dashboard1)

## Key Findings
- 3-point attempts increased 10x from 1980 to 2017
- League scoring declined through the 1990s–2000s before recovering
- Strong correlation between points per game and player efficiency (PER)

## Tools Used
- Python (pandas) — data cleaning
- Tableau Public — visualization

## Data Source
Basketball Reference via Kaggle
